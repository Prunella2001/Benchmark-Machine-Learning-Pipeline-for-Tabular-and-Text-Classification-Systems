# Applied Machine Learning Benchmark Suite

This repository contains two end-to-end machine learning pipelines designed to benchmark classification performance across structured and unstructured data.

The goal of this project is to demonstrate:
- Reproducible ML workflows
- Model benchmarking against baselines
- Feature engineering and data understanding
- Interpretability using modern ML techniques
- Decision-making under real-world constraints (class imbalance, trade-offs, scalability)

---

## 📂 Projects Included

### 1. Toxic Comment Classification (NLP)
A text classification system designed for **content moderation use cases**, using TF-IDF and Logistic Regression.

Key focus:
- Class imbalance handling
- Precision–recall trade-offs
- Threshold tuning for moderation systems

---

### 2. Higgs Boson Classification (Tabular ML)
A high-dimensional physics dataset used to simulate **signal vs background detection in particle physics experiments**.

Key focus:
- Non-linear feature interactions
- Tree-based modeling (LightGBM)
- Model interpretability using SHAP

---

## 🧠 Key Skills Demonstrated

- End-to-end ML pipeline design
- Exploratory Data Analysis (EDA) for model selection
- Supervised learning (classification)
- Handling class imbalance
- Model evaluation (ROC-AUC, Precision, Recall, F1)
- Interpretability (feature importance, SHAP)
- Threshold optimization

Each project is self-contained in its folder.

```bash

```

2. Run the notebook end to end
